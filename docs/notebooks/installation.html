InstallationÔÉÅ

Clone git repositoryÔÉÅ

[ ]:

git clone https://github.com/labstructbioinf/pLAST.git
cd pLAST

conda / mamba environmentÔÉÅ

First, install the mamba environment using the core-env.yml file from the main pLAST repository:
mamba env create -n plast -f core-env.yml
You can also use conda:
conda env create -n plast -f core-env.yml

Alternatively, install the following dependencies manually:

  • python >=3.9, <=3.12

  • pip >= 25.3

  • hmmer >= 3.4

  • mmseqs2 >= 17.b804f

  • prodigal >= 2.6.3

Once all required packages are installed, activate the environment:
mamba activate plast
or
conda activate plast
If you plan to use Jupyter notebooks, install the jupyter package as well:
mamba install jupyter
or
conda install jupyter If you use VS Code and only need notebook support through the Python extension, install just the ipykernel package: mamba install ipykernel
or
conda install ipykernel

pipÔÉÅ

[ ]:

pip install plast

or

[ ]:

pip install git+https://github.com/labstructbioinf/pLAST
